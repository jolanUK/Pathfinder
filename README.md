# Pathfinder - Documentation

Pathfinder is a Laravel/Livewire/Filament application that enhances the ability for mental health outreach programmes to manage courses and participants.

## Technical Information

The application uses the following:

* Laravel v12
* Livewire v4
* Filament v5

This is supplemented by a set of carefully-considered third-party packages.

* Spatie Laravel Settings

## Requirements

It's recommended to have a local development environment set up for this. Laravel Valet or Herd are your best options.
